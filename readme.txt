== Description ==

Several hymns for traditional Christian worship. Learn to sing with letters and Melody.

== Installation ==

1. Upload the plugin Christian Hymns files to the `/wp-content/plugins/christian-hymns` 
directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. No menu plugins clique sobre 'Christian Hymns'
4. The Christian Hymins Plugin is constantly powered by new music tracks. Whenever 
we include new music, we will leave you a new file for you to download and install 
in your Plugins. 
Download the database with the hymns, and then upload the file to complete the 
installation. 
Unzip the files.
